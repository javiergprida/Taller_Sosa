# Taller Sosa - High-Conversion Landing Page

Este proyecto es una Landing Page moderna, orientada a la conversión y altamente optimizada (Mobile-First, SEO Local y Rendimiento) para un taller mecánico integral ubicado en CABA.

## Objetivos del Proyecto
* **Alta Conversión:** Diseño enfocado en derivar el tráfico al Call To Action (WhatsApp).
* **Rendimiento:** Desarrollo 100% en Vanilla HTML/CSS/JS (cero dependencias). 
* **Seguridad (Hardening):** Validación estricta y sanitización del lado del cliente.
* **SEO Local:** Implementación de etiquetas semánticas y Schema.org LocalBusiness JSON-LD.
